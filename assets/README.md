# 🟩 Dossier assets/
- Ressources visuelles – EBrigade OS
- Identité Légion étrangère

---

### 🎯 Mission du dossier
>Le dossier assets/ regroupe toutes les ressources visuelles utilisées par LeBrigade OS :  
- bannières ASCII  
- blasons stylisés  
- thèmes Cinnamon personnalisés  
- éléments graphiques du TUI  

>Ces ressources servent à renforcer l’identité visuelle OPS / RENS / COMSEC (fictive) du projet.

---

### 🧩 Structure interne
```bash
assets/
 ├── ascii/          # Logos, bannières, blasons ASCII
 └── themes/         # Thèmes Cinnamon (GTK, icônes, couleurs)
```

---

### 🖼️ Contenu typique

##### 📁 ascii/
- bannière LeBrigade  
- blason ASCII  
- séparateurs et cadres pour le TUI  
- logos stylisés pour les modules OPS/RENS/COMSEC  

##### 🎨 themes/
- thèmes GTK personnalisés  
- icônes adaptées  
- palettes vert/rouge façon console militaire  
- fichiers .css Cinnamon
