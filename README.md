### 🎯 Objectif du projet
- EBrigade OS est une distribution Linux personnalisée, basée sur Linux 22 (Xia) Cinnamon Edition, conçue pour être administrée, configurée et déployée via une série de scripts `Bash` développés sur mesure.
- Ce projet s’inscrit dans une esthétique OPSRENS / COMSEC, inspirée de l’univers de la Légion étrangère.

---

### 🧩 Architecture du projet
```text
EBrigade/
```

---

### ⚙️ Fonctionnalités prévues
- 🔧 Scripts d’installation automatisée  
- 🎨 Personnalisation complète de Cinnamon (thèmes, icônes, raccourcis)  
- 🛡️ Module COMSEC (renforcement, audit, logs)  
- 🧭 Menu TUI EBrigade pour piloter les scripts  
- 📦 Système modulaire de fonctionnalité ndépendant  
- 🟩 Identité visuelle Légion Étrangère intégrée dans le TUI
